# Healthcare Patient Analysis Dashboard

## 📊 Project Overview

This project analyzes hospital patient data to identify readmission trends and optimize healthcare resources.

## 🔍 Key Features

* Patient Readmission Analysis
* Average Length of Stay by Department
* Department Performance Insights
* Interactive Power BI Dashboard

## 🛠 Tools Used

* Python (Pandas, Matplotlib)
* Power BI
* Data Analysis

## 📁 Files Included

* hospital_data.csv
* analysis.ipynb
* dashboard.png

## 📈 Output

An interactive dashboard providing insights into hospital operations.
